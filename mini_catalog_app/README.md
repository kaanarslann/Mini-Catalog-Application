# Mini Catalog Application

A Flutter application that demonstrates a mini e-commerce catalog. Users can browse products, click a product to view details, and manage a shopping cart.

**Flutter Version**: 3.44.1

## Getting Started

### Prerequisities
- Install Flutter SDK (https://docs.flutter.dev/get-started/install)
- Make sure you have an emulator/device or Chrome

### Installation
1. Clone the repository:
    git clone https://github.com/kaanarslann/Mini-Catalog-Application.git
    cd Mini-Catalog-App

2. Get dependencies:
    flutter pub get

3. Run the application:
    flutter run -d chrome (for web)
